# Texas TEKS — IT Support & Services (Readable List)

Source: 19 TAC Chapter 130, Subchapter K (2015 Adopted)  
143 standards across 3 courses

---

## §130.302 — Principles of Information Technology (Grade 9-10, 1 credit)

### c.1 Professional Standards / Employability Skills
- **c1A** Identify and demonstrate work behaviors that enhance employability (attendance, attire, clean workspace, pride, flexibility, initiative)
- **c1B** Employ effective verbal and nonverbal communication skills
- **c1C** Employ effective reading and writing skills
- **c1D** Solve problems and think critically
- **c1E** Demonstrate leadership skills and function effectively as a team member
- **c1F** Identify and implement proper safety procedures
- **c1G** Demonstrate planning and time-management skills (storyboarding, project management — initiate, plan, execute, monitor, close)

### c.2 Employment Opportunities in IT
- **c2A** Identify job opportunities and accompanying duties and tasks
- **c2B** Research careers of personal interest with required education, skills, and experience
- **c2C** Describe the functions of resumes and portfolios

### c.3 Evolving and Emerging Technologies
- **c3A** Identify and describe functions of various evolving and emerging technologies
- **c3B** Send and receive text/file attachments using electronic methods (email, bulletin boards, instant messaging)
- **c3C** Demonstrate effective Internet search strategies (keywords, Boolean logic, search engines)
- **c3D** Identify the various components of a Uniform Resource Locator (URL)
- **c3E** Test acquired information from the Internet for accuracy, relevance, and validity
- **c3F** Explain issues concerning computer-based threats (viruses, malware, hacking)
- **c3G** Explain Internet safety issues (identity theft, online predators, cyber-bullying, phishing)

### c.4 Hardware Components
- **c4A** Identify major hardware components and their functions
- **c4B** Use available reference tools as appropriate
- **c4C** Connect and use a variety of peripheral devices (mouse, keyboard, microphone, camera, printer)

### c.5 Software Knowledge
- **c5A** Differentiate between systems and application software
- **c5B** Identify and explain major OS fundamentals (disk operations, GUI components, hardware drivers)
- **c5C** Explain the purpose of file types across software products
- **c5D** Demonstrate use of computer numbering systems and internal data representation (hex values)
- **c5E** Compare and contrast open source and proprietary software
- **c5F** Explain use of system management tools
- **c5G** Apply proper file management techniques (create, name, organize, copy, move, delete files)
- **c5H** Use appropriate file protection and security
- **c5I** Explain the process for discovering, quarantining, and removing viruses from a computer system

### c.6 Network Systems Analysis
- **c6A** Identify hardware associated with networking (servers, routers, switches, connectors)
- **c6B** Identify and describe types of networks (peer-to-peer, LAN, WAN, wireless, Ethernet)
- **c6C** Identify functions of network operating systems
- **c6D** Explain troubleshooting techniques for network connection issues

### c.7 Word Processing Technology
- **c7A** Identify terminology associated with word-processing software
- **c7B** Edit text documents (pagination, white space, tab settings, font style/size/color)
- **c7C** Create professional documents (memos, technical manuals, proposals) using advanced features

### c.8 Spreadsheet Technology
- **c8A** Identify terminology associated with spreadsheet software
- **c8B** Use numerical content to perform mathematical calculations
- **c8C** Use functions to produce documents (budget, payroll, statistical tables, checkbook register)
- **c8D** Identify, generate, and describe comma separated value (CSV) files
- **c8E** Create and analyze spreadsheets with advanced features (lookup tables, nested IF, subtotals, charts)
- **c8F** Perform sorting, searching, and data filtering

### c.9 Computer Programming Concepts
- **c9A** Identify the function of compilers and interpreters
- **c9B** Explain the difference between compilers and interpreters
- **c9C** Identify various computer languages and their use in software development
- **c9D** Recognize data representation in software development (string, numeric, character, integer, date)
- **c9E** Identify and explain the concept of algorithms
- **c9F** Describe the flow of a structured algorithm including linear and iterative instructions (flowcharts)

### c.10 Database Technology
- **c10A** Identify terminology associated with database software and functions
- **c10B** Explore the application of databases
- **c10C** Identify and explain the purpose and elements of a query language
- **c10D** Identify and explain the purpose of fields and records
- **c10E** Describe the process of constructing a query including multiple search parameters

### c.11 Presentation Management Technology
- **c11A** Identify terminology and functions of presentation software
- **c11B** Create, save, edit, and produce presentations with advanced features (links, hyperlinks, audio, graphics)

### c.12 Design and Web Publishing Techniques
- **c12A** Identify terminology associated with web page development and interactive media
- **c12B** Identify and explain design elements (typeface, color, shape, texture, space, form)
- **c12C** Identify and explain design principles (unity, harmony, balance, scale, contrast)
- **c12D** Identify and explain common HTML elements (tags, stylesheets, hyperlinks)
- **c12E** Create a web page with links, graphics, and text using design principles

### c.13 Legal and Ethical Procedures
- **c13A** Explain and demonstrate ethical use of technology and online resources
- **c13B** Adhere to intellectual property laws
- **c13C** Explain intellectual property laws (copyright, trademarks, patents) and consequences
- **c13D** Examine the consequences of plagiarism
- **c13E** Identify and explain unethical practices (hacking, online piracy, data vandalism)
- **c13F** Demonstrate ethical use of online resources including citation of source

## §130.303 — Computer Maintenance (Grade 10-12, 1 credit)
*Prerequisite: Principles of Information Technology*

### c.1 Professional Standards / Employability Skills
- **c1A** Employ effective reading and writing skills
- **c1B** Employ effective verbal and nonverbal communication skills
- **c1C** Solve problems and think critically
- **c1D** Demonstrate leadership skills and function effectively as a team member
- **c1E** Identify and implement proper safety procedures
- **c1F** Demonstrate understanding of legal and ethical responsibilities in IT
- **c1G** Demonstrate planning and time-management skills (project management — initiate, plan, execute, monitor, close)

### c.2 Employment Opportunities
- **c2A** Identify job opportunities and accompanying duties and tasks
- **c2B** Examine the role of certifications, resumes, and portfolios in the IT profession

### c.3 Academic Skills Application
- **c3A** Demonstrate communication skills with individuals from varied cultures (workers, management, customers)
- **c3B** Interpret documentation (schematics, drawings, charts, technical manuals, bulletins)

### c.4 Computer Hardware Technologies
- **c4A** Explain the fundamentals of microprocessor theory
- **c4B** Define the use of Boolean and Binary logic in computer technologies
- **c4C** Explain theories of magnetism, electricity, and electronics as related to computer technologies
- **c4D** Explain proper troubleshooting techniques related to computer hardware
- **c4E** Differentiate among digital and analog input and output electronics theory
- **c4F** Explain the relationships relative to data-communications theory
- **c4G** Describe the architecture of various computer systems
- **c4H** Describe function of computer components (CPUs, storage devices, peripheral devices)
- **c4I** Explain computer system environmental requirements and related control devices
- **c4J** Identify new and emerging technologies affecting computer technology

### c.5 Major Computer Components
- **c5A** Identify purpose and function of components (CPU, motherboard, sockets, chipsets, BIOS)
- **c5B** Identify how mobile devices (PDAs, cell phones) operate
- **c5C** Identify how mobile devices connect and share data
- **c5D** Demonstrate understanding of error messages and symptoms of hardware failures
- **c5E** Research interrupt sequences and beep codes
- **c5F** Identify priorities and interrupts at the system level

### c.6 Operating System Design
- **c6A** Explain the fundamentals of an operating system
- **c6B** Compare and contrast different operating systems
- **c6C** Identify the operating systems of mobile devices

### c.7 Software Installation and Configuration
- **c7A** Identify operational features and proper terminology related to computer software systems
- **c7B** Evaluate application software packages
- **c7C** Verify that software is properly licensed prior to installation
- **c7D** Differentiate between software licensing types (SaaS, single-user, per-seat, enterprise, freeware, shareware, open-source)
- **c7E** Explain proper troubleshooting techniques related to computer software

### c.8 Network Connections
- **c8A** Explain the fundamentals of network connections and interface requirements
- **c8B** Explain steps required to install and configure a computer on a network
- **c8C** Identify the steps to troubleshoot network connectivity

## §130.304 — Computer Maintenance Lab (Grade 10-12, 1 credit)

### c.1 Professional Standards / Employability Skills
- **c1A** Demonstrate work behaviors that enhance employability (attendance, attire, workspace, voice, pride)
- **c1B** Demonstrate positive personal qualities (flexibility, open-mindedness, initiative, willingness to learn)
- **c1C** Employ effective reading and writing skills
- **c1D** Employ effective verbal and nonverbal communication skills
- **c1E** Solve problems and think critically
- **c1F** Demonstrate leadership skills and function effectively as a team member
- **c1G** Identify and implement proper safety procedures
- **c1H** Demonstrate understanding of legal and ethical responsibilities in IT
- **c1I** Demonstrate planning and time-management skills (project management — initiate, plan, execute, monitor, close)

### c.2 Academic Skills Application
- **c2A** Complete work orders for repair and installation
- **c2B** Estimate supplies, materials, and labor costs for work orders
- **c2C** Locate and interpret documentation (schematics, drawings, charts, technical manuals, bulletins)

### c.3 Tools, Equipment, and Materials
- **c3A** Demonstrate safe use of equipment in computer technologies (hand and power tools)
- **c3B** Employ available reference documentation (tools, materials, Internet sources)
- **c3C** Demonstrate proper handling and disposal of environmentally hazardous materials
- **c3D** Research new and emerging technologies affecting computer technology

### c.4 Trade Concepts in Simulated Work
- **c4A** Use electronic test equipment to measure current, voltage, power, and resistance
- **c4B** Describe digital circuits and bus design
- **c4C** Demonstrate operational features and terminology related to computer systems
- **c4D** Demonstrate proper usage of computer system components (CPU, BIOS, ROM, RAM)
- **c4E** Troubleshoot computer peripheral devices

### c.5 Hardware Design and Maintenance
- **c5A** Assemble and install a basic computer system
- **c5B** Install and configure computer components (printers and peripherals)

### c.6 Troubleshooting Skills
- **c6A** Diagnose error messages and symptoms of hardware failures
- **c6B** Research and identify interrupt sequences and beep codes
- **c6C** Identify priorities and interrupts at the system level
- **c6D** Test a system using diagnostic tools and software
- **c6E** Diagnose problems in operating systems
- **c6F** Differentiate between hardware and software failure
- **c6G** Update Basic Input/Output System (BIOS)
- **c6H** Demonstrate hard drive maintenance procedures (defrag, scan, clear caches)
- **c6I** Gather information from the user
- **c6J** Repair malfunctioning hardware systems
- **c6K** Reinstall software as needed
- **c6L** Demonstrate system backup and recovery
- **c6M** Restore a system to various states (safe mode, previous states)
- **c6N** Demonstrate knowledge of operating system design (operation and maintenance)
- **c6O** Apply OS knowledge to perform IT support tasks across different operating systems

### c.7 Software Installation and Configuration
- **c7A** Evaluate application software packages and test functionality of a proposed configuration
- **c7B** Verify software is properly licensed prior to installation
- **c7C** Install application and systems software using available resources
- **c7D** Resolve problems with installation (recovery from system error)
- **c7E** Perform software customization as requested
- **c7F** Document all procedures
- **c7G** Install and maintain security software

### c.8 Network Installation and Configuration
- **c8A** Demonstrate understanding of network connection and interface requirements
- **c8B** Install and configure a computer on a network
- **c8C** Verify and troubleshoot network connectivity

### c.9 User Support and Maintenance Planning
- **c9A** Develop a written disaster recovery plan
- **c9B** Develop a written preventive maintenance plan
