## Learning objective

Students will be able to organize ideas and source material, compile them accurately with citations, and write a short healthcare report or summary that uses audience-first structure and plain language.

## Content stub

- **The three verbs as a sequence, not three skills:** *organize* (outline: audience, purpose, key points) → *compile* (gather facts, data, and sources, cite them) → *write* (produce the document)
- **Audience-first structure:** headline / main point → background → findings → recommendations; one idea per paragraph; plain language for patient-facing material
- **Report genres students will actually produce:** incident reports, shift summaries, patient-education handouts, lab write-ups, public-health briefs
- **Citing sources in healthcare writing:** cite original data and guidelines; quote sparingly; never fabricate numbers; flag where the evidence is weak
- **Editing pass before submission:** clarity (does every sentence add something?), precision (are values measurable?), completeness (is the audience's next question answered?)
- **Relationship to c1A and c2F:** c1A sets the clarity/conciseness bar, c2F sets the precision bar — c2D is the production-of-a-written-artifact standard that must meet both

## Assessment stub

- Summary build: given a 5–10 page public-health report, produce a one-page summary for a non-expert audience; peer-scored on audience fit, clarity, completeness, and citation
- Outline-first exercise: given a stack of unordered facts about a patient or a health topic, produce a written outline for a named audience before drafting — instructor feedback on the outline, then the draft
- Incident-report practical: given a simulated adverse event (e.g., medication near-miss), produce a factual, non-blaming incident report that meets organizational reporting requirements

## References

- **AHRQ Health Literacy Universal Precautions Toolkit** — plain-language writing standards for patient-facing reports and summaries.
- **AMA Manual of Style (11th ed.)** — the default style reference for US medical writing, including citation, abbreviation, and numerical conventions.
- **AHRQ / TJC event-reporting formats** — structure and required elements for incident reports, including factual, non-blaming language.
- **ICMJE Recommendations** — source for the audience-first / IMRAD-adjacent structure used in health-sciences writing.
