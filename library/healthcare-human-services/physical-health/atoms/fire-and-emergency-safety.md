## Learning objective

Students will be able to respond to a fire or emergency in a healthcare setting using the RACE and PASS protocols, and identify common emergency codes.

## Content stub

- **RACE:** Rescue anyone in immediate danger, Alarm (pull the pull-station, call code), Contain (close doors/windows), Extinguish or Evacuate — in that order
- **PASS (fire extinguisher):** Pull the pin, Aim at the base of the fire, Squeeze the handle, Sweep side-to-side; know when NOT to fight a fire
- **Oxygen and fire:** O2 doesn't burn but accelerates combustion; shut valves, keep flame/heat away from tubing, know where shutoffs are
- **Emergency codes (vary by facility):** Code Red (fire), Code Blue (cardiac arrest), Code Gray (combative person), Code Pink (infant abduction), Code Silver (active shooter)
- **Evacuation priorities:** ambulatory first, then wheelchair, then bedbound; horizontal (to next smoke compartment) before vertical (stairs)
- **Incident reporting:** document what happened, when, who was involved, what was done — near-misses count too; purpose is system improvement, not blame

## Assessment stub

- Tabletop drill: given a unit map with a fire in room 412, walk through RACE step-by-step with peers assigned roles
- Code quiz: match 10 announcements to the correct code and the correct initial response
- Extinguisher skill: demonstrate PASS on a training extinguisher or simulator; explain when to fight vs. flee
