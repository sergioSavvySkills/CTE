# Digital Technology → Web & Cloud

## Scope (Advance CTE definition)

> *"Careers focused on creating and managing websites and web
> applications, including front-end and back-end development and
> maintenance, as well as overseeing cloud services and infrastructure.
> This field merges software skills with user interface/user experience
> and cloud management expertise, ensuring website functionality,
> security, and cloud efficiency."*

Source: *Guidebook: The Modernized National Career Clusters Framework*,
Advance CTE, April 2025 (p. 5).

**Example programs of study:** Cloud Computing, Web & Digital
Communications, Web Development.

## Provenance & verification status

- Canonical data lives in `web-cloud.csv` alongside this file.
- Researched against **7 states**: CA, TX, FL, NY, PA, OH, IL.
- This sub-cluster spans two domains: **web development** (CIW,
  Certiport, HTML/CSS/JS) and **cloud computing** (AWS, Azure, Google
  Cloud, CompTIA Cloud+). Both domains are well-established in CTE
  with strong state-list presence.
- Heavy cross-cutting with Software Solutions (JavaScript, HTML/CSS
  credentials appear in both). Credentials are tagged accordingly.
- Ohio and Texas have the broadest credential recognition. PA is
  notable for being the only state where W3Schools certs appear on
  an official guide.

## Anchor recommendations

**Three-year pathway stack for HS Web & Cloud:**

1. **Certiport IT Specialist: HTML and CSS** *(Year 1, 9th-10th)* —
   lowest barrier to entry. 36-40 questions, 50 min, ~$100, proctored.
   Ideal first-credential experience. Pairs with Certiport ITS: Cloud
   Computing as a second Year 1 credential if the program covers both
   domains.

2. **CIW Web Foundations Associate** *(Year 2, 10th-11th)* — the
   comprehensive web credential. Covers internet-business + site-
   development + networking. $175 per attempt. 3-4x the Ohio IRC points
   vs. Certiport ITS at 1 pt. This is the credential with the most
   state-list weight for accountability purposes.

3. **AWS Certified Cloud Practitioner** *(Year 3, 11th-12th)* — cloud
   capstone. $100, 65-item proctored CBT, no prerequisites. Dominant
   cloud platform. TX, FL, OH, PA confirmed. 3-year validity gives
   students a durable credential post-graduation. If the program has
   Microsoft Imagine Academy access and teaches Azure instead of AWS,
   **AZ-900** is an equivalent alternative (free vouchers).

## State research notes

- **Texas:** Broadest inclusion. 2025-2030 IBC list includes CIW (all),
  AWS Cloud Practitioner, AWS Solutions Architect, AZ-900, AZ-104,
  Google Cloud ACE, CompTIA Cloud+, Certiport ITS (all), Adobe ACP
  Dreamweaver, Google UX Design, MS-900.

- **Florida:** 2025-26 CAPE list includes CIW, AWS Cloud Practitioner,
  Azure Fundamentals, CompTIA Cloud+, Certiport ITS, Adobe ACP. CAPE
  requires 150 hours of instruction tied to each credential.

- **Ohio:** Most detailed with point values. CIW Web Foundations (4 pt),
  CIW Site Dev (3 pt), CIW Advanced HTML5 (4 pt), CIW JavaScript
  (4 pt), AWS CCP (4 pt), AWS SA Associate (4 pt), AZ-900 (4 pt),
  Google Cloud ACE (6 pt), CompTIA Cloud+ (6 pt), Certiport ITS (1 pt
  each), Adobe ACP Dreamweaver (4 pt), Google UX Design (2 pt),
  MS-900 (4 pt). Note: Certiport ITS at 1 pt makes them low-value for
  accountability relative to CIW (3-4 pt) and AWS/Azure (4 pt).

- **California:** No statewide list. District-level. CIW, AWS, Azure
  commonly used at district discretion.

- **New York:** NYSED does NOT list CIW, AWS, Azure, or Certiport ITS
  on the publicly visible approved page. Not prohibited — just not
  pre-approved at state level. Individual program approval required.

- **Pennsylvania:** PDE guide includes CIW (Associate Design, Site Dev,
  Web Design, JavaScript), AWS CCP, AZ-900, Adobe ACP, W3Schools
  HTML/Python, CodeHS Web Design/Dev. PA is the only state where
  W3Schools appears on an official document.

- **Illinois:** ISBE recommended list not machine-readable. Individual
  confirmations pending.
