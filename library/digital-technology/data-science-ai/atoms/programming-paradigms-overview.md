## Learning objective

Students will be able to define the major programming paradigms and identify which paradigm a given code snippet exemplifies.

## Content stub

- **Procedural:** step-by-step instructions grouped into functions; classic C-style; still how most simple scripts read
- **Object-oriented (OOP):** bundle data + behavior into objects; classes, inheritance; natural for modeling real things
- **Functional:** compose pure functions, avoid mutation; map/filter/reduce; popular in data pipelines (Pandas chain calls)
- **Logic:** declare facts and rules, let the system infer answers (Prolog); used in expert systems
- **Python is multi-paradigm:** supports all four; you'll mostly mix procedural + OOP + functional in AI/DS code

## Assessment stub

- Classify 5 short snippets as procedural / OOP / functional / logic
- Research: find a real ML library and identify which paradigms it uses
- Short answer: why is 'multi-paradigm' an advantage for Python in data science?
