## Learning objective

Students will be able to name the fundamental data types computers work with and identify the appropriate type for a given value.

## Content stub

- **Numeric types:** integers for counts, floats for measurements; why you don't use floats for money
- **Text:** characters and strings; Unicode support; common encodings (UTF-8)
- **Dates and times:** stored as numbers but displayed as formatted strings; timezone complexity
- **Graphics:** bitmap (raster) vs vector; pixel values; file formats (PNG, JPEG, SVG)
- **Audio:** sampled waveforms; sample rate and bit depth; formats (WAV, MP3, FLAC)
- **Boolean:** True/False; used for flags, toggles, decision branches

## Assessment stub

- Classify 10 example values (e.g., 3.14, 'Ada', 2026-04-17, a smiley image) by their data type
- Short answer: why is representing money as a float dangerous?
- Lab: examine a spreadsheet column and identify its data type
