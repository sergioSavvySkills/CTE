## Learning objective

Students will be able to declare and manipulate basic Python data types and predict the result of common type operations.

## Content stub

- **Integers and floats:** int = whole numbers, float = decimals; arithmetic: + - * / // % **
- **Strings:** text in quotes; concatenation with +; indexing with []; f-strings for formatting
- **Booleans:** True and False; comparison operators ==, !=, <, >, <=, >=
- **Lists:** ordered, mutable collection in [ ]; index access, .append(), .pop(), slicing
- **Type conversion:** int('5'), str(42), float('3.14'), bool(0); when implicit coercion happens
- **type() and common errors:** type(x) to inspect; TypeError when you mix incompatible types

## Assessment stub

- Predict the output: `'3' + str(5)` and `3 + int('5')`; explain the difference
- Lab: write 5 lines of Python that use each data type at least once
- Debug: given code with a TypeError, identify and fix the line
