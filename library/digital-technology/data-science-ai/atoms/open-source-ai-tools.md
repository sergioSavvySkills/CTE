## Learning objective

Students will be able to identify major open-source AI tools and describe what each is used for.

## Content stub

- **TensorFlow:** Google's deep-learning framework; production-ready; broad ecosystem
- **PyTorch:** Meta's deep-learning framework; dominant in research; dynamic computation graph
- **Scikit-learn:** classical ML (regression, trees, clustering); the entry point for most data scientists
- **Hugging Face Transformers:** pre-trained NLP and vision models; one-line access to thousands of models
- **Spark ML:** distributed ML on Apache Spark; used when data doesn't fit on one machine
- **License types:** MIT, Apache 2.0, BSD — most AI tools are permissively licensed for commercial use

## Assessment stub

- Research: pick one library and find the GitHub star count, license, and major maintainers
- Compare: TensorFlow vs PyTorch — 3 ways they differ
- Short answer: why has the open-source ecosystem been so important for modern AI progress?
