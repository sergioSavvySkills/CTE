## Learning objective

Students will be able to write Python programs using conditionals, for-loops, and while-loops to control program execution.

## Content stub

- **if / elif / else:** evaluate a boolean condition; branch to the matching block; else is the fallback
- **for loops:** iterate over a sequence: `for x in [1,2,3]:` or `for i in range(10):`
- **while loops:** repeat while a condition remains True; watch for infinite loops; break to exit early
- **Nested control flow:** loops inside loops, or if inside for — used for grids, retries, multi-step logic
- **Common patterns:** accumulator (running total), counter, filter, early-exit with break/continue
- **Event-driven basics:** programs that wait for input and react — a loop + an if is the core pattern

## Assessment stub

- Write a program that asks for 5 numbers and prints the sum and average
- Write a program that counts how many vowels are in a user-entered string
- Debug: a while loop runs forever — identify why and fix
