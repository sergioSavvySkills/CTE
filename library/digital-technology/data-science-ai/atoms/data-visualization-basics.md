## Learning objective

Students will be able to pick the right chart type for a dataset and build readable visuals in a spreadsheet or Python.

## Content stub

- **Chart types by purpose:** bar = compare categories, line = change over time, scatter = correlation, histogram = distribution, pie = shares (sparingly)
- **Good-chart principles:** clear axis labels, honest scales, legible font, single headline question per chart
- **Avoiding misleading visuals:** truncated y-axis, 3D effects, dual axes, overly-small pie slices — common ways to mislead
- **Tools:** Excel/Sheets charts, Python matplotlib/seaborn, Tableau, Power BI, D3 for custom web
- **Box plots, stem/dot plots:** less flashy but powerful for small datasets; show distribution without binning

## Assessment stub

- Pick the right chart: given 4 datasets, name the best chart type and justify
- Lab: create a bar chart and a line chart in a spreadsheet; fix 2 labeling problems
- Critique: find a misleading chart online and identify the technique used
