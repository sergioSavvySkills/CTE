## Learning objective

Students will be able to organize different kinds of data in a spreadsheet, apply built-in functions, and format for readability.

## Content stub

- **Rows and columns:** one row = one record, one column = one attribute; headers in row 1
- **Cell data types:** numbers, text, dates, formulas — watch out for auto-converted dates and leading-zero loss
- **Built-in functions:** SUM, AVERAGE, COUNT, MIN, MAX, IF, VLOOKUP/XLOOKUP, CONCAT
- **Sorting and filtering:** sort by one or more columns; filter to focus on a subset
- **Formatting:** number format (currency, %, date), conditional formatting to highlight outliers
- **Keyboard shortcuts:** arrow keys + Ctrl/Cmd + Shift for fast navigation and selection

## Assessment stub

- Lab: import a 50-row dataset, sort by a numeric column, and apply conditional formatting to outliers
- Write 3 formulas: sum a column, count non-blank cells, look up a value by id
- Troubleshoot: given a CSV where dates imported as numbers, fix the formatting
