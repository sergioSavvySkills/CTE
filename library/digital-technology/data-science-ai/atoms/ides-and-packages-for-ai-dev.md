## Learning objective

Students will be able to set up a Python AI dev environment and use essential packages for ML work.

## Content stub

- **VS Code:** most popular editor; Python and Jupyter extensions; GitHub Copilot integration
- **PyCharm:** JetBrains IDE; deep Python support; paid Pro version for scientific/data features
- **Jupyter Notebooks:** browser-based interactive computing; standard tool for DS exploration; JupyterLab is the modern UI
- **Google Colab:** free hosted Jupyter with GPU access; perfect for class projects
- **Essential packages for ML:** numpy, pandas, matplotlib, scikit-learn, torch or tensorflow, transformers
- **Virtual environments:** venv or conda; one per project; keeps dependencies clean and reproducible

## Assessment stub

- Lab: open Google Colab, install Pandas and Scikit-learn, load the iris dataset, print head
- Compare: VS Code vs Jupyter vs Colab — when would you use each?
- Short answer: why should every AI project live in its own virtual environment?
