## Learning objective

Students will be able to assemble a simple embedded robot or device that uses sensors and actuators for a specific task.

## Content stub

- **Plan the build:** sketch the robot; list required parts (microcontroller, sensors, motors, chassis, battery)
- **Safety first:** static discharge precautions; polarity when connecting; never short power to ground
- **Wiring:** use a breadboard for prototyping; jumper wires to microcontroller pins; check schematic twice before powering on
- **Testing components individually:** blink an LED; read a sensor value; drive a motor — verify each works before combining
- **Chassis and mounting:** 3D-print or laser-cut a chassis; secure components so they don't shake loose
- **Iterate:** build in stages: locomotion works → add sensor → add AI decision → add output

## Assessment stub

- Lab: assemble a Raspberry Pi or Arduino with one sensor and one output (LED or buzzer); verify it powers on
- Document: produce a wiring diagram with component list
- Troubleshoot: given a robot that powers on but motors don't spin, describe your diagnostic steps
