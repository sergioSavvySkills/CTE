# Digital Technology → Software Solutions

## Scope (Advance CTE definition)

> *"Careers involved in the development and maintenance of software
> solutions, built on computer science and encompassing programming,
> application development, and front-end and back-end software
> development. This Sub-Cluster also includes developing emerging
> technologies such as augmented and virtual reality, the Internet of
> Things, distributed ledger technologies, and quantum computing. It
> involves creating the underlying code and systems that power
> applications, designing user interfaces for optimal user experience,
> and building server-side technologies that process data behind the
> scenes."*

Source: *Guidebook: The Modernized National Career Clusters Framework*,
Advance CTE, April 2025 (p. 5).

**Example programs of study:** App Design, Computer Simulation, Game
Development, Programming.

## Provenance & verification status

- Canonical data lives in `software-solutions.csv` alongside this file.
- Researched against **7 states**: CA, TX, FL, NY, PA, OH, IL.
- This is the most credential-rich Digital Technology sub-cluster. Heavy
  cross-cutting with Web & Cloud (JavaScript, HTML/CSS) and Data Science
  & AI (Python). Credentials appearing in multiple sub-cluster CSVs are
  tagged with `cross_cutting_tags`.
- AP Computer Science A and AP CSP are universally recognized across all
  7 states — the only credentials in this sub-cluster with that status.

## Anchor recommendations

**Four-credential anchor stack for a HS Software Solutions pathway:**

1. **AP Computer Science A** — the undisputed gold standard. Java-based,
   universally recognized, score 3+ yields college credit. Revised
   2025-26 (Unit 9 removed). Every state recognizes it. The anchor
   every other credential stacks against.

2. **PCEP - Entry-Level Python Programmer** — $59, 30 questions, 40
   minutes. Most affordable proctored cert in the space. One semester
   of Python is sufficient prep. TX IBC, OH 3 pts. Stacks to PCAP and
   PCPP. Python dominance in AP CSP and data science makes this a
   natural complement to AP CSA's Java.

3. **Certiport IT Specialist: Java or Software Development** — the
   year-round proctored industry cert that directly aligns to AP CSA
   content. 50-min CBT, ~$100, can be administered at school-based
   testing centers any time (not May-only like AP). Good for students
   who want an industry cert in addition to the AP score.

4. **Unity Certified User: Programmer** — the game dev / AR-VR anchor.
   C# + Unity engine. TX IBC, OH 4 pts. High student engagement.
   Directly aligns to Game Development and Computer Simulation programs
   of study. The VR Developer variant adds AR/VR coverage.

## State research notes

- **California:** No statewide approved certification list. Credential
  selection is district-level. AP CSA/CSP universally recognized.

- **Texas:** 2025-2030 IBC list (523 certs) includes Certiport ITS
  (all tracks), Oracle Java Foundations, Unity Certified User, Apple
  Swift, CodeHS, PCEP. Broadest inclusion.

- **Florida:** 2025-26 CAPE list includes Certiport ITS, Oracle Java,
  Unity Certified User Programmer, CIW exams, AP CSA/CSP. PCEP verify.

- **New York:** Decentralized. NYSED list skews toward higher-level
  Oracle/professional certs. Does NOT list Certiport ITS, CIW, Unity,
  or PCEP. Python Institute listed only at PCPP (professional) level.

- **Pennsylvania:** PDE guide includes CIW JavaScript, Oracle certs,
  W3Schools Python/HTML, Azure Fundamentals, CodeHS (coming 25-26).
  NOCTI/NBS state-level assessments also available.

- **Ohio:** Most detailed list with point values. Certiport ITS (1 pt
  each), PCEP (3 pt), Unity (4 pt each), CIW (3-4 pt), Oracle Java
  Foundations (6 pt), Oracle Java SE 8 (9 pt), Oracle Professional
  (12 pt). 3-year IRC framework re-evaluation underway.

- **Illinois:** ISBE recommended list not machine-readable. Flexible
  and continuously updated. Individual confirmations pending.
