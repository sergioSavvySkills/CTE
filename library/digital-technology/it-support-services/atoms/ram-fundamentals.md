## Learning objective

Students will be able to identify RAM form factors, describe DDR generation differences, and determine correct RAM upgrades for a given system.

## Content stub

- **DIMM vs SODIMM:** full-size desktop DIMMs vs. small-outline laptop DIMMs; not interchangeable
- **DDR generations:** DDR3/DDR4/DDR5 — voltage, speed (MT/s), bandwidth improvements; backward-incompatibility
- **ECC vs non-ECC:** error-correcting code RAM for servers and workstations; detects and corrects single-bit errors
- **Dual-channel:** installing RAM in matched pairs doubles memory bus width; performance benefit for integrated graphics
- **Capacity and speed:** how much RAM is enough for common tasks; XMP/EXPO profiles for overclocking
- **Reading a RAM spec:** decoding labels like 'DDR4-3200 16GB CL16 1.35V DIMM'

## Assessment stub

- Upgrade scenario: user has 8 GB DDR4-2666 and wants to upgrade — what must they verify before buying?
- Lab: use CPU-Z or Speccy to read RAM specs from a live system and fill in a spec sheet
- Short answer: why can't you install a DDR4 stick in a DDR5 motherboard?
