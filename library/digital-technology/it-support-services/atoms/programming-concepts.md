## Learning objective

Students will be able to define core programming vocabulary including variables, functions, objects, and libraries, and recognize these constructs in sample code.

## Content stub

- **Variables and identifiers:** naming rules, data types in code, scope (local vs. global), constants
- **Functions/procedures:** defining and calling functions; parameters and return values; reusability principle
- **Object-oriented basics:** class as a blueprint; object as an instance; properties and methods; encapsulation concept
- **Libraries and APIs:** importing pre-written code; standard library vs. third-party packages; API calls
- **Error handling:** exceptions and try/catch; defensive coding; logging errors for troubleshooting
- **Reading code samples:** recognizing Python, JavaScript, and Bash snippets; understanding enough to maintain scripts

## Assessment stub

- Code reading: given a 15-line Python script, identify three variables, one function, and one loop
- Vocabulary match: match 10 programming terms to definitions without notes
- Short answer: why do programmers use functions instead of copying the same code multiple times?
