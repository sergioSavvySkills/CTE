## Learning objective

Students will be able to use built-in spreadsheet functions to perform calculations for budgets, invoices, and financial summaries.

## Content stub

- **SUM and AVERAGE:** =SUM(range), =AVERAGE(range); using ranges vs. individual cells; AutoSum shortcut
- **COUNT functions:** =COUNT (numbers only), =COUNTA (any value), =COUNTIF (conditional count)
- **IF function:** =IF(condition, value_if_true, value_if_false); nested IF; common business uses
- **VLOOKUP/XLOOKUP:** looking up values in a table; exact vs. approximate match; XLOOKUP as modern replacement
- **Financial functions:** =PMT for loan payments; =SUM for totals; =ROUND for currency rounding
- **Practical document types:** invoice template, budget tracker, grade calculator; applying functions to real forms

## Assessment stub

- Invoice lab: build a 10-line invoice template using SUM, IF for discount logic, and ROUND for totals
- VLOOKUP exercise: given a product catalog table, use VLOOKUP to populate prices from a product ID
- Short answer: what does the FALSE parameter in VLOOKUP specify?
