## Learning objective

Students will be able to classify programming languages by type and abstraction level, and match each category to its common use cases in IT work.

## Content stub

- **Low-level languages:** machine code and assembly; directly control hardware; used in firmware, drivers, embedded systems
- **High-level compiled languages:** C, C++, Rust, Go; compiled to machine code; fast execution; used in OS and systems programming
- **Interpreted and scripting:** Python, JavaScript, Ruby, Bash; run via interpreter; rapid development; used in automation and web
- **Managed/bytecode languages:** Java, C#; compiled to bytecode then JIT-compiled; platform-independent; enterprise use
- **Query languages:** SQL for databases; not general-purpose; declarative style
- **Markup and style:** HTML, CSS, XML, JSON; not Turing-complete; structure and presentation of data

## Assessment stub

- Language classification: sort 12 languages into the correct categories
- Use-case matching: match 5 IT tasks (automate file backup, build a web page, query a database, write a device driver, create an Android app) to the best language choice
- Short answer: why would a DevOps engineer choose Python over C++ for an automation script?
