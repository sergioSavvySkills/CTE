## Learning objective

Students will be able to write valid HTML using common tags to structure a web page with headings, paragraphs, links, images, and lists.

## Content stub

- **HTML document structure:** DOCTYPE, html, head, body; metadata (title, charset, viewport meta tag)
- **Text elements:** h1–h6 headings, p paragraphs, strong, em, br, hr; semantic meaning vs. visual appearance
- **Links and images:** a href for hyperlinks (absolute vs. relative URLs); img src and alt attribute; why alt is required
- **Lists:** ul (unordered), ol (ordered), li items; nested lists
- **Tables:** table, tr, th, td; thead, tbody; when to use tables (tabular data only, not layout)
- **Forms basics:** form, input (text/email/checkbox/radio/submit), label, textarea, select; action and method attributes

## Assessment stub

- Coding exercise: hand-code a 4-section HTML page (bio, skills list, links, and a contact form) without any CSS
- Tag identification: given 20 lines of HTML, identify the tag and purpose of each element
- Short answer: why is the alt attribute on an img tag important for both accessibility and SEO?
