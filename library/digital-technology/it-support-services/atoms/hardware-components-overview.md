## Learning objective

Students will be able to identify the major internal and external hardware components of a computer system and describe the function of each.

## Content stub

- **CPU:** the processor — fetches, decodes, executes instructions; cores and clock speed
- **RAM:** volatile working memory; holds running programs and open data
- **Storage:** HDD vs SSD; where data lives when the computer is off
- **Motherboard:** the main circuit board connecting all components; buses and slots
- **Power supply:** converts AC wall power to DC voltages the components need
- **Ports and expansion:** USB, HDMI, PCIe slots, audio jacks; how components communicate externally

## Assessment stub

- Label-the-diagram: given a PC interior photo, identify 10 components by name
- Component function quiz: match each component to a one-sentence description
- Short answer: why does a computer need both RAM and storage?
