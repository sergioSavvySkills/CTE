## Learning objective

Students will be able to install and configure a wireless access point, set security parameters, and verify client connectivity.

## Content stub

- **WAP placement:** signal range, interference sources, channel overlap; 2.4 GHz vs. 5 GHz coverage trade-offs
- **SSID configuration:** visible vs. hidden SSID; naming conventions; guest networks
- **Security settings:** WPA2-Personal vs. WPA2-Enterprise; WPA3; disabling WEP/WPS
- **DHCP and IP assignment:** router-assigned vs. static IPs for wireless clients; reservation by MAC address
- **Channel selection:** non-overlapping channels (1/6/11 for 2.4 GHz); auto vs. manual selection; avoiding neighbor interference
- **Testing and verification:** checking signal strength, running a speed test, verifying security protocol in client adapter settings

## Assessment stub

- Lab: configure a consumer router with a new SSID, WPA3 password, and guest network isolation
- Signal mapping: use a Wi-Fi analyzer app to map signal strength across a room
- Short answer: why is WEP considered insecure and what should replace it?
