## Learning objective

Students will be able to describe how data travels across a network, identify the layers of the OSI model, and explain the role of common networking protocols.

## Content stub

- **What is a network:** LAN, WAN, MAN, PAN; nodes, links, and switches; why networks exist
- **OSI model:** 7 layers (Physical → Data Link → Network → Transport → Session → Presentation → Application); each layer's job; how layers encapsulate data
- **TCP/IP model:** 4-layer practical model that maps to OSI; dominant in real networks
- **Protocols overview:** HTTP/HTTPS, FTP, SMTP, DNS, DHCP — each protocol's purpose and port
- **Packets and frames:** how data is broken into packets; headers add addressing; frames add MAC addresses at layer 2
- **Network devices:** hub vs. switch vs. router vs. access point — where each operates in the OSI stack

## Assessment stub

- OSI labeling: given a scenario (e.g., browser request), identify which layer handles each step
- Protocol matching: match 10 protocols to their port numbers and descriptions
- Short answer: what is the difference between a switch and a router?
