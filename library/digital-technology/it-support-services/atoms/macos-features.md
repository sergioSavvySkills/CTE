## Learning objective

Students will be able to navigate macOS, identify its key features and built-in utilities, and describe how it differs from Windows in common support scenarios.

## Content stub

- **macOS interface:** Finder, Dock, Menu Bar, Spotlight, Mission Control; window management gestures
- **System Preferences / Settings:** key panes: Displays, Sound, Network, Users & Groups, Security & Privacy, Software Update
- **Built-in utilities:** Terminal (bash/zsh), Activity Monitor, Disk Utility, Console, Network Utility, Keychain Access
- **File system and paths:** APFS volume structure; /Users /Applications /Library; case sensitivity; .DS_Store files
- **macOS support tools:** Safe Mode, Recovery Mode (Cmd+R), Internet Recovery, First Aid in Disk Utility; SIP
- **macOS vs Windows for support:** different driver model, App Store/DMG installs, Gatekeeper, iCloud integration

## Assessment stub

- Navigation task: use Spotlight to open Activity Monitor and identify the top memory-consuming process
- Short answer: a macOS app won't open and shows 'unidentified developer' — what setting controls this and how do you bypass it?
- Comparison: list three macOS equivalents to Windows tools (e.g., Activity Monitor = Task Manager)
