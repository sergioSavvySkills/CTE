## Learning objective

Students will be able to create strong passwords, explain password policies, and configure common password policy settings in Windows.

## Content stub

- **Password strength:** length vs. complexity; NIST SP 800-63B guidance — length matters more than special chars
- **Password policies:** minimum length, complexity rules, expiration, history, lockout threshold and duration
- **Password managers:** how they work; vault encryption; master password risk; browser integration
- **Common attacks:** brute force, dictionary attack, credential stuffing, rainbow tables; how length defeats each
- **Default credentials:** why factory defaults are exploitable; requirement to change on first use
- **Windows password policy:** Local Security Policy, Fine-Grained Password Policy (domain); configuring lockout threshold

## Assessment stub

- Policy configuration lab: configure a Windows local password policy to match a provided security standard
- Password strength comparison: given 5 passwords, rank them by strength and explain the ranking
- Short answer: why does NIST no longer recommend mandatory 90-day password expiration?
