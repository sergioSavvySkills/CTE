## Learning objective

Students will be able to apply the core design principles — contrast, alignment, repetition, proximity, hierarchy, and balance — to evaluate and improve digital layouts.

## Content stub

- **Contrast:** making important elements stand out; color contrast for accessibility (WCAG AA ratio); size contrast
- **Alignment:** grid systems; aligning elements to create order; left/center/right alignment choices
- **Repetition (Consistency):** repeating colors, fonts, and shapes to create visual cohesion; style guides and design systems
- **Proximity:** grouping related items close together; separating unrelated items; gestalt principle
- **Hierarchy:** leading the eye from most to least important; using size, color, and placement to rank content
- **Balance:** symmetrical vs. asymmetrical balance; visual weight; avoiding one-sided layouts

## Assessment stub

- Principle identification: analyze a provided web page and identify one example of each principle
- Redesign exercise: given a poorly designed flyer, apply all 6 principles to improve it
- Short answer: how does visual hierarchy guide a user's eye on a web page?
