## Learning objective

Students will be able to describe the responsibilities of a system administrator and distinguish the sysadmin role from help desk, network, and security roles.

## Content stub

- **Sysadmin scope:** managing servers, OS, user accounts, software deployment, backups, monitoring — the full stack
- **Sysadmin vs. help desk:** help desk handles end-user incidents; sysadmin handles infrastructure and escalated issues
- **Common sysadmin tasks:** provisioning/deprovisioning accounts, patching servers, capacity planning, on-call response
- **Infrastructure documentation:** network diagrams, runbooks, change logs, asset inventory — sysadmin as documentation owner
- **Automation in sysadmin work:** scripting repetitive tasks; configuration management (Ansible, Group Policy); why automation reduces error
- **Career path:** help desk → junior sysadmin → senior sysadmin → infrastructure architect; certifications (MCSA, RHCSA, VCP)

## Assessment stub

- Role comparison: create a table distinguishing help desk, sysadmin, network admin, and security analyst by daily tasks
- Scenario triage: given 10 IT incidents, classify each as help desk vs. sysadmin responsibility
- Short answer: why is documentation considered one of the most important sysadmin skills?
