## Learning objective

Students will be able to create, format, and save word processing documents using a modern word processor, applying text formatting, paragraph styles, and document structure.

## Content stub

- **Interface overview:** ribbon/toolbar layout; document view modes; zoom; ruler; status bar
- **Text formatting:** font face, size, bold/italic/underline, color, highlight, clear formatting
- **Paragraph formatting:** alignment (left/center/right/justify), line spacing, indentation, paragraph spacing
- **Lists and bullets:** bulleted lists, numbered lists, multilevel lists; list indent/outdent
- **Undo, save, and file formats:** Ctrl+Z, autosave, Save vs. Save As; .docx vs. .pdf vs. .odt; compatibility mode
- **Basic navigation:** Ctrl+Home/End, Find & Replace (Ctrl+H), word count, spell check

## Assessment stub

- Formatting exercise: format a provided plain-text document to match a given formatted sample
- Short answer: what is the difference between character formatting and paragraph formatting?
- Practical: save a document in both .docx and .pdf format and explain when each is appropriate
