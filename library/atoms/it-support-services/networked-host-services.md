## Learning objective

Students will be able to describe the services that servers provide on a network, including file sharing, authentication, web, email, and print services.

## Content stub

- **File services:** SMB/CIFS for Windows shares, NFS for Unix/Linux; mapping drives; permissions on shared folders
- **Web services:** HTTP/HTTPS servers (IIS, Apache, nginx); how a browser request is served; SSL certificates
- **Email services:** SMTP for sending; IMAP and POP3 for retrieval; spam filtering; MX records
- **Authentication services:** LDAP, Active Directory, RADIUS; how a domain login works; single sign-on concept
- **Print services:** print server spooling jobs; network printers vs. local shared printers; IPP protocol
- **Remote services:** SSH, RDP, VNC; Telnet (legacy/insecure); how remote access is authenticated and encrypted

## Assessment stub

- Service-port matching: given 10 network services, identify the protocol and port number for each
- Scenario: a user can log into their laptop but not access a shared drive — list three possible causes
- Short answer: what is the difference between IMAP and POP3 for email retrieval?
