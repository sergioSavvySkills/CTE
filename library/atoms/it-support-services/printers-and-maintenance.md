## Learning objective

Students will be able to describe printer technologies, perform routine maintenance tasks, and troubleshoot common printing problems.

## Content stub

- **Printer technologies:** laser (toner, drum, fuser), inkjet (cartridge types), thermal (receipt printers), impact (dot-matrix legacy)
- **Laser print process:** processing → charging → exposing → developing → transferring → fusing → cleaning (7-step EP)
- **MFDs:** multifunction devices combining print/scan/copy/fax; network-shared vs. locally connected
- **Driver and spooler:** print driver types (PCL vs PostScript); Windows print spooler; clearing stuck print jobs
- **Routine maintenance:** toner/ink replacement, drum cleaning, paper feed rollers, calibration, firmware updates
- **Connectivity:** USB, Ethernet, Wi-Fi Direct, cloud print (Google Cloud Print successor, AirPrint, Mopria)

## Assessment stub

- Maintenance lab: replace toner cartridge, run a cleaning page, and run a calibration print
- Troubleshoot scenario: user gets 'paper jam' but no paper is visible — walk through the diagnosis
- Short answer: what is the fuser and what failure symptom does it cause when it fails?
