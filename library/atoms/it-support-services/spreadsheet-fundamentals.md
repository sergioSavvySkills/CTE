## Learning objective

Students will be able to enter and organize data in a spreadsheet, apply basic formatting, and navigate common spreadsheet interface features.

## Content stub

- **Interface overview:** rows, columns, cells, cell references (A1 notation); Name Box; Formula Bar; sheet tabs
- **Data entry:** typing text vs. numbers vs. dates; AutoFill for series; data validation basics
- **Cell formatting:** number formats (currency, percentage, date), text alignment, cell borders and fill color, wrap text
- **Basic arithmetic:** entering formulas (=A1+B1); order of operations; relative vs. absolute references ($A$1)
- **Sorting and filtering:** single-column sort; multi-level sort; AutoFilter; filtering by value
- **Saving and file formats:** .xlsx vs. .csv vs. .ods; when to use each; protecting a workbook with a password

## Assessment stub

- Data entry lab: enter a provided dataset (20 rows), apply currency and date formatting, and sort by date
- Formula exercise: write formulas to calculate total, average, and percentage in a budget spreadsheet
- Short answer: what is the difference between a relative and an absolute cell reference?
