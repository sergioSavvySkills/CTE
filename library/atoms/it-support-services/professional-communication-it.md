## Learning objective

Students will be able to employ effective verbal, nonverbal, reading, and writing skills when communicating with end users, team members, and customers in an IT support setting.

## Content stub

- **Verbal communication:** active listening, avoiding jargon with non-technical users, confirming understanding
- **Nonverbal communication:** tone, body language, attentiveness during customer interactions
- **Written communication:** ticket notes, email etiquette, technical documentation for both internal and customer-facing use
- **Reading for meaning:** interpreting user descriptions of problems, reading error messages carefully, parsing technical documentation
- **Difficult situations:** de-escalation, handling frustrated users, escalating appropriately
- **Cross-cultural awareness:** communicating with colleagues and customers from varied backgrounds

## Assessment stub

- Role-play: student plays help-desk tech fielding a call from a frustrated non-technical user; peer scores communication using a rubric
- Writing task: draft a clear ticket note summarizing a multi-step troubleshooting session
- Reading task: given a long user complaint, identify the actual problem in one sentence
