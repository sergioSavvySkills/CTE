## Learning objective

Students will be able to safely install a CPU, thermal solution, RAM, and PCIe card into a motherboard, following ESD precautions and verifying POST success.

## Content stub

- **ESD safety:** anti-static wrist straps, mats, bags; touching the case; humidity; why ESD matters
- **CPU installation:** locating the alignment notch/arrow; ZIF socket (Intel LGA vs AMD AM5); seating force; protecting pins
- **Thermal paste:** cleaning old paste, applying new paste (pea-sized or spread method); heatsink clip/screw-down
- **RAM seating:** aligning the notch; pressing until clips click; dual-channel slot order
- **PCIe card installation:** selecting the correct slot (x16, x8, x4, x1); securing with screw; driver install after boot
- **Post-install verification:** first POST; BIOS shows CPU, RAM; boot to OS; checking Device Manager for unknown devices

## Assessment stub

- Hands-on lab: install CPU, heatsink, and RAM into a practice board (or use a decommissioned machine)
- Step-by-step checklist: given a scenario, list all safety steps before opening a PC case
- Short answer: what does it mean if the PC powers on but immediately shuts off after a CPU install?
