## Learning objective

Students will be able to describe cloud computing characteristics, compare deployment models, and identify common cloud services used in IT support environments.

## Content stub

- **Cloud characteristics:** on-demand self-service, broad network access, resource pooling, rapid elasticity, measured service (NIST 5 essentials)
- **Deployment models:** public cloud (AWS/Azure/GCP), private cloud, hybrid cloud, community cloud — trade-offs
- **Service models:** IaaS/PaaS/SaaS — reviewed from IT support perspective; who manages what in each
- **Common cloud services:** storage (S3, Blob), compute (EC2, VMs), identity (Azure AD), backup, CDN
- **Cloud in IT support:** cloud backup, cloud-managed devices, SaaS ticketing, cloud print; why IT techs must understand cloud
- **Cost and licensing:** pay-as-you-go vs. reserved instances; cloud sprawl; FinOps basics

## Assessment stub

- Deployment model scenario: given 3 org profiles, recommend public/private/hybrid and justify
- Short answer: a company's file server runs in AWS — which cloud service model is that?
- Cost exercise: compare monthly cost of on-prem server vs. equivalent cloud VM for a small office
