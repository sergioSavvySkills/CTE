## Learning objective

Students will be able to build a complete, styled, multi-section web page combining HTML structure with CSS visual design.

## Content stub

- **CSS introduction:** inline vs. internal vs. external stylesheet; selector, property, value syntax; linking CSS to HTML
- **CSS selectors:** element, class (.), ID (#); specificity basics; descendant selectors
- **Box model:** content, padding, border, margin; how box-sizing affects layout; inspecting with DevTools
- **Typography and color in CSS:** font-family, font-size, color, background-color; Google Fonts import; rem vs. px
- **Layout basics:** display block vs. inline vs. inline-block; flexbox for simple layouts; centering techniques
- **Responsive design intro:** meta viewport tag; % widths; max-width; media query concept (not required to write)

## Assessment stub

- Web page project: build a personal IT portfolio page with a header, about section, skills list, and contact section — styled with external CSS
- DevTools inspection: open any website, use browser DevTools to modify the CSS of one element and take a screenshot
- Short answer: what does the CSS box model include and how does padding differ from margin?
