## Learning objective

Students will be able to define core web design vocabulary, identify the roles on a web team, and describe how websites are structured and delivered.

## Content stub

- **How websites work:** client (browser) requests a URL; DNS resolves to IP; server returns HTML/CSS/JS; browser renders
- **Web design vs. web development:** design = visual/UX; front-end dev = HTML/CSS/JS; back-end dev = server/database; full-stack
- **Key terminology:** wireframe, mockup, prototype, landing page, CTA, responsive design, viewport, above the fold
- **Interactive media types:** static vs. dynamic websites; forms, carousels, video embeds, maps, chatbots
- **File types in web design:** PNG vs. JPG vs. SVG vs. WebP; web fonts (WOFF2); icon libraries
- **Hosting and deployment:** web hosting, domain registrar, DNS A records; FTP/SFTP upload; CMS vs. coded site

## Assessment stub

- Vocabulary quiz: match 15 web design terms to definitions
- Website anatomy: label a provided website screenshot with 10 web design terms
- Short answer: what is the difference between a wireframe and a mockup?
