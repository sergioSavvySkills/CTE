## Learning objective

Students will be able to differentiate Windows Home, Pro, Enterprise, and Education editions by feature set and identify the appropriate edition for a given scenario.

## Content stub

- **Windows 10/11 editions:** Home, Pro, Pro for Workstations, Enterprise, Education — licensing model differences
- **Pro vs. Home features:** BitLocker, Remote Desktop host, domain join, Group Policy, Hyper-V — only in Pro+
- **Enterprise and Education:** volume licensing, Windows Update for Business, LTSC channel, extra security features
- **Windows version history:** XP → Vista → 7 → 8/8.1 → 10 → 11; support lifecycle; why old versions are a risk
- **Upgrade paths:** Home to Pro upgrade; in-place upgrade vs. clean install; activation and product keys
- **OEM vs. retail license:** OEM tied to hardware; retail transferable; how to check activation status

## Assessment stub

- Edition matching: given 5 workplace scenarios, identify the correct Windows edition and justify
- Short answer: a school wants Group Policy management — which edition do they need?
- Activation check: on a lab machine, find and record the Windows edition, build number, and activation status
