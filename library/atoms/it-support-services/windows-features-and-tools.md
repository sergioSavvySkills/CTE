## Learning objective

Students will be able to use built-in Windows administrative tools to view system information, manage tasks, configure settings, and access help resources.

## Content stub

- **Task Manager:** processes, performance graphs, startup items, services tab; killing hung processes
- **MMC snap-ins:** Computer Management, Device Manager, Disk Management, Event Viewer, Local Users and Groups
- **System Information (msinfo32):** hardware summary, components, software environment; useful for support calls
- **Control Panel vs. Settings:** legacy vs. modern UI; which settings still live in Control Panel; how to open both
- **Resource Monitor and Performance Monitor:** detailed CPU/disk/network/memory usage; creating baselines; identifying bottlenecks
- **Windows Update and servicing:** checking update status, cumulative vs. feature updates, pausing updates, update troubleshooting

## Assessment stub

- Lab: open Task Manager and identify the top 3 CPU-consuming processes; end a non-responsive process
- Tool scavenger hunt: use msinfo32, Device Manager, and Event Viewer to complete a system info worksheet
- Short answer: what is the difference between the Services tab in Task Manager and the Services MMC snap-in?
