## Learning objective

Students will be able to create polished professional documents using styles, headers/footers, tables, and images in a word processor.

## Content stub

- **Styles and themes:** applying heading styles (H1/H2/H3); body text style; modifying a style; why styles matter for consistency
- **Headers, footers, and page numbers:** inserting header/footer; different first page; page number formatting
- **Tables:** inserting tables, merging cells, formatting borders, sorting table data, table styles
- **Images and objects:** inserting images, wrapping text, resizing, alt text for accessibility
- **Page layout:** margins, orientation, paper size, columns, page breaks vs. section breaks
- **Track changes and comments:** enabling track changes; accepting/rejecting; adding comments; collaboration workflow

## Assessment stub

- Document creation project: produce a two-page professional report with title, heading styles, a table, and a footer with page numbers
- Short answer: why should you use heading styles rather than manually bolding and increasing font size?
- Peer review exercise: use track changes to mark up a classmate's document with suggested edits
